REQUIREMENT
1. node.js - https://nodejs.org/en/

INSTRUCTION
1. Navigate to this folder with a shell (visual studio, just open in terminal)
2. Type in "npm install" to allow user input
3. Run by typing in "node main.js"
4. Enter in sentence(s). Copy and paste works best. DO NOT paste multiple paragraphs 
with a new line inbetween

------------------------------------------------------------------------------------
When adding into the vocabulary list, first check "used_recipe_searches" to see if the 
search has already been made. Find a suitable search and add in only the recipe instruction
paragraph. Remember to add the search that you made into the list so that we don't overfill
in similar searches.
